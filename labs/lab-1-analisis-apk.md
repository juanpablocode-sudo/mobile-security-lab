# 📱 Lab 1 — Análisis básico de APK

## 🎯 Objetivo

Analizar una aplicación Android para identificar posibles riesgos de seguridad.

---

## 📦 Aplicación analizada

Nombre: app.apk  
Tipo: Aplicación simple  

---

## 🔍 Análisis

### 📄 Permisos encontrados

- INTERNET  
- ACCESS_NETWORK_STATE  

### ⚠️ Observaciones

La aplicación solicita acceso a red, lo cual puede implicar comunicación con servidores externos.

---

## 🧠 Conclusión

El análisis básico permite identificar posibles riesgos y entender el comportamiento de la aplicación.

---

## 🔐 Aplicación en seguridad

Este tipo de análisis es clave para detectar aplicaciones maliciosas o vulnerables.
